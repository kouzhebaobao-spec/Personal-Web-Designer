# 个人主页定制业务 · AI工作流 SOP

## 一、业务概述

用AI辅助，为求职者快速定制专业个人主页，从接单到交付全流程约30-60分钟。

---

## 二、完整工作流

### 第1步：客户下单
- 将「客户信息收集表单.html」部署到在线平台（推荐：GitHub Pages / Netlify，均免费）
- 在小红书、朋友圈、BOSS直聘等渠道发布，引导客户填写
- 客户提交后，你收到邮件通知（需配置表单后端，推荐：Formspree.io）

---

### 第2步：整理客户信息（5分钟）
收到客户信息后，将以下内容整理成一段文字发给Claude：

```
客户信息：
- 姓名：XXX
- 目标职位：XXX
- 城市：XXX
- 联系方式：XXX
- 一句话介绍：XXX
- 个人简介：XXX
- 教育背景：XXX
- 工作经历：XXX
- 技能：XXX
- 项目经历：XXX
- 风格偏好：XXX / 主色调：XXX
- 补充说明：XXX
```

---

### 第3步：发给Claude的指令模板（直接复制使用）

```
请根据以下客户信息，生成一个完整的个人求职主页HTML文件：

[粘贴客户信息]

要求：
1. 风格：[现代简约/创意活力/科技感]
2. 主色调：[颜色代码]
3. 包含：导航栏、Hero区、关于我、技能、工作经历、项目、联系方式
4. 确保移动端适配
5. 代码完整可直接运行
```

---

### 第4步：Claude生成主页（15-20分钟）
Claude会输出完整HTML代码，保存为 `客户姓名_个人主页.html`

---

### 第5步：审核与调整（10分钟）
- 用浏览器打开HTML文件预览效果
- 检查信息是否准确
- 如需调整，告知Claude具体修改点

---

### 第6步：交付（5分钟）

**方式A：直接发HTML文件**
- 将HTML文件通过微信/邮件发给客户
- 附上《部署教程.pdf》（告知客户如何上线）

**方式B：帮客户部署上线（加收费用）**
- 推荐使用 GitHub Pages（免费）
- 部署后给客户一个可访问的网址

## 四、常见问题处理

**Q：客户想要自己的照片怎么办？**
A：让客户提供照片，将图片转为Base64嵌入HTML，或部署时放到同一文件夹

**Q：客户要求修改怎么办？**
A：告知Claude具体改哪里，一般5分钟内搞定

**Q：客户不会部署怎么办？**
A：提供录屏教程，或收费帮助部署（建议额外收100-200元）
